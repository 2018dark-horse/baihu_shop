# baihu_shop（ilong）

## Getting Started
flutter packages get

## lib目录下项目结构
* components:	公共组件
* config:	公共配置
* model:	数据模型
* routes:	路由配置
* tools:	公共工具
* views:	界面

### views目录
* main:根界面
* home:商城主页
* profile:个人中心

#### 功能目录
* pages:界面
* components:组件
* routes:子路由配置
* bloc:逻辑代码
